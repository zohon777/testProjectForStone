Для запуска приложения вам нужно запустить локальный сервер php,
для этого с помощью командной строки зайдите в папку проекта, затем введите команду

php artisan serve

после этого у вас будет доступен локальный сервер по адресу http://127.0.0.1:8000/
заходите по этому адресу, далее вы увидите очень простую форму, здесь вы можете проводить тестирование.

Данный вариант приложения был сделан в ограниченный срок, так что я не стал делать проверки на правильность ввода даных,
так же внешний вид можно улучшить, но так как я стремился показать функционал, не обращал внимание на внешний вид.
